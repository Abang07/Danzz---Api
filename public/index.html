API ini gratis dan tidak perlu API key. Langsung akses dari browser, Postman, atau kode kamu.

  1. Pilih kategori endpoint di atas lalu klik untuk expand
  2. Klik ▶ Coba untuk test langsung di sini
  3. Atau klik Salin URL untuk copy endpoint nya
  4. Terima response JSON dan gunakan datanya

Base URL:

https://danzz-api.online

Downloader — TikTok:

GET /api/downloader/tiktok?url=https://vt.tiktok.com/ZS9qqmN16/

Stalk — TikTok:

GET /api/stalker/stalk-tiktok?username=khaby.lame
